Gyro Bluetooth Mouse
This is an Android application that turns your smartphone into a Bluetooth mouse for your computer using the phone's built-in gyroscope.

Features
Gyroscope Control: Move your phone through the air to control the mouse cursor.

Bluetooth HID: Connects to any computer (Windows, macOS, Linux) without needing any special server software on the computer side. It acts like a standard Bluetooth mouse.

Simple UI: Provides buttons for left and right clicks.

Landscape Mode: Locked to landscape for a more comfortable and natural grip.

How to Build
You can build this project using Android Studio or by using the included GitHub Actions workflow. After pushing this code to a GitHub repository, the APK will be automatically built and available for download under the "Actions" tab.
